ENGLISH

Introduction
In this update, changes have been made to the JavaScript on my site. The main aim is to improve readability and add support for new YouTube link types. Below you'll find a summary of the changes made to each part.

Added link checking for the 2 new link types in the 'if' of line 16
Changed the conditional block else... if... block for a switch... block, adding a total of 11 cases plus a default case
A stylization has been added to the invalid link in the first if... conditional structure. The stylization consists of making the text of the "error" div bold and red
Conclusion
In this update, improvements were made to the style of the elements and the readability of the code. With these updates, the site has finalized the implementation of YouTube support.

Next steps:
In the next update, the final process of implementing Vimeo in JavaScript is expected to take place, which will allow Vimeo videos to be displayed and played on the site.
